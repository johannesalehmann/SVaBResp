# SVaBResp

Shapley-Value-based responsibility

## Installation

First obtain a version of the [Rust toolchain](https://rust-lang.org/learn/get-started/).

Then navigate to the root of the repository, i.e. the directory that contains this README. Execute the following command
to build SVaBResp:

    cargo build --package svabresp-cli --release --bin svabresp-cli

You can now invoke SVaBResp using

    ./target/release/svabresp-cli

Use the ```--help``` flag to see available arguments.
